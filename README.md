# 📄主要用途

* 自动关闭＆重启计算机

# ❓如何使用

1. 双击“Aute_Shutdown_版本.exe”；
2. 点击“下一步；
3. 点击“下一步；
4. 再次点击“下一步；
5. 设置提示的时间（分钟）；
6. 设置提示后关机＆重启的时间（秒）；*提示后需要确认才会开始等待
7. 选择操作（关机＆重启）；
8. 等、等、等……；
9. 在弹出的窗口点击“确定”。

# 📱使用软件

* Python（主程序）
* EasyGUI（界面）
* Pyinstaller（打包）
