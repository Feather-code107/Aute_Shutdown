# ✏️使用 | USE

1. [下载](https://github.com/Feather-code107/Aute_Shutdown/releases)软件；
2. 以管理员身份运行程序；
3. 单击“下一步”；
4. 认真阅读警告后单击“下一步”；
5. 再次单击“下一步”；
6. 设置提示时间（分钟）；
7. 设置提示后关机＆重启的时间（秒）；
8. 等……；
9. 一会儿后，在弹出的窗口中单击“确定”；
10. 电脑将在你设定的时间（第6步）后自动关机。
